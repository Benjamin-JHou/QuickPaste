无需账号、无需下载、无需数据线。打开网页，设备之间秒级传输文字和图片。

打开快贴 → 创建房间

快贴是怎么工作的

三步实现跨设备传输——无论手机是 iPhone 还是安卓，电脑是 Windows 还是 Mac，都行。

1
创建一个房间
在任意浏览器的快贴页面，点击「创建快贴房间」，自动生成 6 位房间号。可以直接分享二维码，或者复制链接发给其他人。
2
在另一台设备上加入
在另一台手机/平板/电脑上，扫房间二维码，或者输入 6 位房间号，即可立即加入同一个房间。
3
开始互传
输入文字或粘贴图片，实时发送到所有已连接设备。P2P 直传，数据不过服务器。

三种加入房间的方式

根据场景选择最方便的方式。

A
扫二维码
用相机对准 Host 设备屏幕上的二维码，大多数手机相机可直接识别并跳转到快贴页面加入房间。
B
分享链接
点击「复制分享链接」，通过微信、邮件、AirDrop 等任意方式发送。对方打开链接即自动加入房间。
C
手动输入房间号
在另一台设备上点击「扫码或输入房间号」，输入 Host 屏幕上显示的 6 位字母数字组合即可。

安装到桌面 / 主屏幕

像原生 App 一样使用，无需从应用商店下载。

📱 iPhone / iPad（Safari）

  1. 在 Safari 中打开快贴
  2. 点击底部分享按钮（方框向上箭头）
  3. 向下滚动，点击「添加到主屏幕」
  4. 点击添加——完成！

🤖 Android（Chrome）

  1. 在 Chrome 中打开快贴
  2. 地址栏右侧会出现「安装应用」按钮，点击即可
  3. 或在菜单 ⋮ 中选择「安装应用」
  4. 安装后桌面出现快贴图标

💻 Windows / Mac（Chrome/Edge）

  1. 在 Chrome 或 Edge 中打开快贴
  2. 地址栏右侧点击安装图标
  3. 快贴以独立窗口打开，无浏览器边框

🖥️ 桌面浏览器（Firefox/Safari）

  1. 在任意浏览器打开快贴即可
  2. 添加到书签栏方便随时使用
  3. 所有功能在浏览器标签页中正常运行

可以传什么？

任何能在浏览器里展示的内容。

💬
文字和消息
输入即发，适合网址、地址、电话号码、快速备注。
🖼️
图片和照片
从相册选图或直接粘贴（Ctrl+V / ⌘V），自动压缩到约 500KB。
📋
剪贴板粘贴
在一台设备上复制图片，在另一台设备按 Ctrl+V，自动发送。
🤸
多设备同时
不限数量，所有设备加入同一房间，实时同步内容。
🔒
P2P 加密
数据在设备之间直接传输，连服务器都无法读取内容。
⚡
免账号
打开就能用，不需要注册、登录、验证手机号。

常见问题

直接回答你的疑问。

问
没有网络能用吗？
两台设备都需要联网来建立 P2P 连接。建立后数据直接在设备间传输，用流量也可以。
问
数据存在哪里？会被保存吗？
不保存任何数据。快贴没有数据库，所有消息只存在浏览器内存里。关闭标签页 = 数据永久消失。
问
房间号是什么格式？
6 位字母数字组合（去掉了容易混淆的 0/1/I/L/O）。每个房间号随机生成、互不重复。
问
一个房间能连多少台设备？
不限数量。所有设备组成 mesh 网络，房间内所有人都能看到所有消息。
