How QuickPaste works

Three steps to share across all your devices — no matter if it's iPhone, Android, Windows, or Mac.

1
Create a room
Open QuickPaste in any browser. Tap Create a room. A 6-character room code is generated — or share the QR code directly.
2
Open on another device
On your other phone/tablet/computer, open the same URL or scan the QR code. Tap Scan or enter room ID. You join the same room instantly.
3
Start sharing
Type text or paste images — they're sent directly to all connected devices in real time via P2P. No uploads, no servers touching your data.

Three ways to join a room

Pick whichever is most convenient for your situation.

A
Scan the QR code
Point your camera at the host's QR code. Most phone cameras can read QR codes directly — just point and tap the notification.
B
Share the link
Tap Copy share link on the host device. Send it via WeChat, email, AirDrop, or any app. Opening the link opens QuickPaste and joins the room automatically.
C
Type the room code
On another device, tap Scan or enter room ID and type the 6-character code shown on the host's screen.

How QR code scanning works

What the QR code actually contains, and what happens when you scan it.

📦 What's encoded in the QR code

The QR code displayed in the room is just a URL like https://…/index.html#room=ABC234. The 6-character room ID after #room= is the only data — no text content, no images, no personal info. Anyone with this URL can join that specific room.

🔍 Scanning the QR on your phone

On iOS, open the Camera app (or Code Scanner) and point at the QR. On Android, use Google Lens or the built-in camera. The phone decodes the URL and offers a banner link — tap it to open in your default browser, which will load QuickPaste and auto-join the room.

🔐 Privacy of the QR code

The QR only reveals the room ID. Once you join, content transferred through the room is sent device-to-device (WebRTC), not through any server. The QR code can't be used to "peek" at messages — it only opens a join page.

📲 What if your phone has no camera or QR scanner?

Use method B or C above: tap Copy share link on the host and send the URL via chat / email / AirDrop, or manually type the 6-character room code on the second device.

Install as an app

Add to your home screen — opens like a native app, works offline for the UI.

📱 iPhone / iPad (Safari)

  1. Open QuickPaste in Safari
  2. Tap the Share button (□↑ arrow)
  3. Scroll down, tap "Add to Home Screen"
  4. Tap Add — done!

🤖 Android (Chrome)

  1. Open QuickPaste in Chrome
  2. Tap the install prompt in the address bar, or tap the menu ⋮ → Install app
  3. Confirm — it appears as an app icon

💻 Windows / Mac (Chrome/Edge)

  1. Open QuickPaste in Chrome or Edge
  2. Click the install icon in the address bar
  3. QuickPaste opens in its own window, no browser chrome

🖥 Desktop (Firefox/Safari)

  1. Open QuickPaste in your browser
  2. Bookmarks bar shortcut works great too
  3. The PWA runs as a tab, all features work normally

FAQ

Common questions, direct answers.

Q
Does it work without internet?
Both devices need an internet connection to establish the P2P connection initially. Once connected, data flows directly between devices. Works over mobile data too.
Q
Is my data stored anywhere?
No. QuickPaste has no database, no server-side storage. Messages only exist in browser memory. Close the tab = gone forever.
Q
What's the room code format?
6 characters, uppercase letters and numbers (0, 1, I, L, O excluded to avoid confusion). Each room code is randomly generated and unique.
Q
How many devices can join one room?
Unlimited. All devices connect in a mesh network — everyone sees everything sent to the room.
