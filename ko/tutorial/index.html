快貼 작동 원리

세 단계로 크로스 디바이스 전송 — iPhone, 안드로이드, Windows, Mac 상관없이.

1
방 만들기
임의의 브라우저에서 快貼 열기. 방 만들기 탭하면 6자리 방 번호 자동 생성 — QR 코드 공유 또는 링크 복사 가능.
2
다른 기기에서 접속
다른 폰/태블릿/PC에서 QR 코드 스캔하거나 6자리 방 번호 입력하면 같은 방에 즉시 접속.
3
공유 시작
텍스트 입력하거나 이미지 붙여넣기 — P2P로 연결된 모든 기기에 실시간 전송. 서버 거치지 않음.

방에 접속하는 3가지 방법

상황에 맞는 가장 편리한 방법을 선택하세요.

A
QR 코드 스캔
호스트 기기 화면의 QR 코드를 카메라로 스캔. 대부분의 스마트폰 카메라가 QR 코드 직접 인식 후 快貼 페이지로 이동.
B
링크 공유
링크 복사 탭 → 카톡, 이메일, AirDrop 등 아무 앱으로 전송. 상대방이 링크 열면 자동 접속.
C
방 번호 직접 입력
다른 기기에서 QR 또는 방 번호 입력 → 호스트 화면의 6자리 문자+숫자 조합 입력.

앱으로 설치하기

홈 화면에 추가하면 네이티브 앱처럼 작동, 앱스토어 불필요.

📱 iPhone / iPad (Safari)

  1. Safari에서 快貼 열기
  2. 하단 공유 버튼 (□↑ 화살표) 탭
  3. 내려서 "홈 화면에 추가" 탭
  4. 추가 — 완료!

🤖 Android (Chrome)

  1. Chrome에서 快貼 열기
  2. 주소창 오른쪽 앱 설치 버튼 탭
  3. 또는 메뉴 ⋮ → 앱 설치
  4. 홈 화면에 아이콘 생성됨

💻 Windows / Mac (Chrome/Edge)

  1. Chrome 또는 Edge에서 快貼 열기
  2. 주소창 오른쪽 설치 아이콘 클릭
  3. 독립 창으로 실행 — 브라우저 없이

🖥️ 데스크톱 (Firefox/Safari)

  1. 원하는 브라우저에서 快貼 열기
  2. 북마크 바에 추가하면 편리
  3. 탭으로도 모든 기능 정상 작동

무엇을 보낼 수 있나요?

브라우저에서 표시할 수 있는 모든 것.

💬
텍스트 & 메시지
즉시 입력·전송. URL, 주소, 전화번호, 짧은 메모에 적합.
🖼️
이미지 & 사진
갤러리에서 선택하거나 붙여넣기(Ctrl+V / ⌘V). 자동 압축 ~500KB.
📋
클립보드 붙여넣기
한 기기에서 이미지 복사 → 다른 기기에서 Ctrl+V — 버튼 불필요, 자동 전송.
🤸
복수 기기 동시
제한 없음. 모든 기기가 mesh 네트워크로 연결 — 방 안所有人都能看到所有内容.
🔒
P2P 암호화
데이터가 기기 간 직접 전송. 서버도 내용을 볼 수 없음.
⚡
계정 불필요
페이지 열면 바로 사용. 가입, 로그인, 전화번호 인증 불필요.
🌐
크로스 플랫폼 전송
브라우저만 있으면 어디서든 — iOS, Android, Windows, macOS, Linux. 앱 다운로드 불필요.
🔐
프라이버시 전송
종단 간 암호화. 로그 불록, 서버 저장 없음. 전송 내용은 내 기기 사이에만 존재.
🕐
임시 전송
데이터는 세션 동안만 존재. 방을 닫으면 모든 내용이 자동으로 사라지며, 어디에도 저장되지 않음.

자주 묻는 질문

바로 답변.

Q
인터넷 없으면 작동하나요?
초기 P2P 연결 수립 시 두 기기 모두 인터넷 필요. 연결 후에는 데이터가 직접 전송되며, 모바일 데이터도 사용 가능.
Q
데이터가 저장되나요?
저장 안 됨. 快貼은 데이터베이스 없음. 모든 메시지는 브라우저 메모리에만 존재. 탭 닫으면 영구 삭제.
Q
방 번호 형식은?
6자리 대문자+숫자 (혼동 쉬운 0, 1, I, L, O 제외). 각 방 번호는 무작위 생성·고유.
Q
한 방에 최대 몇 대?
제한 없음. 모든 기기가 mesh 네트워크 연결 — 방 안所有人都能看到所有发送的内容.
